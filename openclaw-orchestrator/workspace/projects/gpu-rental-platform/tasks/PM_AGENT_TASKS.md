# PM Agent（产品部）任务清单

## 任务 1：GPU算力租赁市场调研
**优先级**: P0 | **截止时间**: 第1周末

### 调研内容
1. **市场规模**
   - 全球GPU云服务市场规模（2024-2026数据）
   - 中国GPU租赁市场增速与预测
   - 驱动因素：AI大模型训练、自动驾驶、AIGC爆发

2. **客户画像**
   - AI创业公司：需要大规模训练算力，价格敏感
   - 高校/科研机构：预算有限，注重GPU型号多样性
   - 独立开发者：按需使用，追求易用性
   - 中大型企业：稳定性优先，需要专属集群

3. **付费模式研究**
   - 按小时计费：灵活但单价高
   - 包月/包年：价格优惠，锁定客户
   - 竞价实例：利用闲置资源，价格波动
   - 预留实例：保证资源，适合长期项目

## 任务 2：竞品分析矩阵
**优先级**: P0 | **截止时间**: 第2周中

### 分析对象
**国内竞品**：
- AutoDL（算力共享平台龙头）
- 趋动云（OrionX，GPU虚拟化）
- 矩池云（Matpool，性价比路线）
- 阿里云 PAI-EAS
- 腾讯云 GPU 云服务器
- 华为云 ModelArts

**海外竞品**：
- Vast.ai（去中心化算力市场）
- RunPod（GPU云，开发者友好）
- Lambda Cloud（深度学习专用）
- CoreWeave（AI基础设施，估值$90亿+）

### 对比维度
| 维度 | 说明 |
|------|------|
| 价格 | A100/H100/4090等型号每小时价格 |
| GPU型号 | 支持的GPU种类与库存 |
| 可用性 | SLA承诺、实际可用性 |
| 易用性 | 注册→创建实例→SSH连接的操作步数 |
| 生态 | 预装框架、模板镜像、社区 |
| 增值服务 | 数据集、模型市场、技术支持 |

## 任务 3：产品定位与PRD
**优先级**: P0 | **截止时间**: 第2周末

### 产品定位
- 目标市场定位（与竞品的差异化）
- 核心价值主张
- 品牌定位语

### PRD内容
1. **功能清单（MVP）**
   - 用户注册/登录（含企业认证）
   - GPU实例浏览与创建
   - 实例生命周期管理（启动/停止/重启/释放）
   - 实时计费与余额充值
   - SSH/Jupyter Lab 远程访问
   - 监控面板（GPU利用率、温度、显存）
   - 账单与发票

2. **功能清单（V2.0）**
   - 多实例集群管理
   - 自定义镜像上传
   - 数据集挂载
   - 定时任务/自动伸缩
   - API SDK

3. **定价策略**
   - 各GPU型号定价建议
   - 首单优惠/邀请返利
   - 企业客户阶梯定价
