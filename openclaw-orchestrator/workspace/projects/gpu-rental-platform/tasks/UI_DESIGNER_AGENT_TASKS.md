## 任务 2：Web端设计（Figma）
**优先级**: P0 | **截止时间**: 第4周

### 页面设计清单

#### 2.1 官网/营销页
- Hero Section: 核心价值主张 + CTA按钮
- GPU型号展示: 参数对比卡片
- 价格方案: 定价表格 + 计算器
- 客户案例/评价
- FAQ区域
- Footer: 链接、社交媒体、备案信息

#### 2.2 用户控制台
- **仪表盘 (Dashboard)**
  - 运行中实例卡片
  - GPU利用率环形图
  - 本月消费趋势折线图
  - 快捷操作（新建实例、充值）

- **实例管理页**
  - 实例列表（表格视图）
  - 实例状态标签（运行中/已停止/创建中）
  - 操作按钮（SSH连接、Jupyter、VNC）
  - 批量操作工具栏

- **实例创建向导**
  - Step 1: 选择GPU型号（卡片式选择器）
  - Step 2: 选择镜像（Ubuntu + 预装框架）
  - Step 3: 配置SSH密钥
  - Step 4: 确认订单

- **监控详情页**
  - 实时GPU利用率波形图
  - 温度/功耗/显存仪表盘
  - 网络流量图表
  - 告警事件时间线

- **账单中心**
  - 余额卡片（充值按钮）
  - 消费明细表格（日期筛选、导出）
  - 发票申请表单

- **设置页**
  - 个人资料表单
  - SSH密钥管理（添加/删除）
  - API Token管理
  - 通知偏好设置

#### 2.3 响应式适配
- 桌面端（≥1440px）: 完整布局
- 平板端（768px-1024px）: 折叠侧边栏
- 移动端（<768px）: 单列布局，底部导航

## 任务 3：设计系统 (Design System)
**优先级**: P0 | **截止时间**: 第4周

### 组件库
| 组件 | 变体 | 说明 |
|------|------|------|
| Button | Primary/Secondary/Danger/Ghost + 4尺寸 | 全站按钮规范 |
| Input | Default/Error/Success/Disabled | 表单输入 |
| Card | Default/Hover/Selected | 实例卡片、GPU卡片 |
| Table | Default/Sortable/Selectable | 实例列表、账单表格 |
| Badge | Status（running/stopped/pending） | 状态标签 |
| Modal | Confirm/Form/Detail | 弹窗组件 |
| Toast | Success/Error/Warning/Info | 全局提示 |
| Chart | Line/Bar/Donut/Gauge | 监控图表 |

### 设计规范
- **间距系统**: 4px基准（4/8/12/16/24/32/48）
- **圆角**: 4px（小）/ 8px（中）/ 12px（大）
- **阴影**: 3层级（卡片/悬浮/弹窗）
- **动画**: 150ms/250ms/350ms 三种时长
- **断点**: sm(640) / md(768) / lg(1024) / xl(1280) / 2xl(1536)
