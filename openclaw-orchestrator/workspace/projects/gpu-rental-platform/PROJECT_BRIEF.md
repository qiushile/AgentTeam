# GPU算力租赁行业智能体系统 - 项目简报

**发起时间**: 2026-06-25
**发起人**: 世乐（邱世乐）
**项目状态**: 任务分发阶段

## 项目概述
构建一套完整的GPU算力租赁行业智能体系统，覆盖从市场调研、产品开发到运营维护的全生命周期。

## Agent 部门与职责分工

### 1. PM Agent (产品经理)
- GPU租赁市场调研
- 竞品分析（国内外主流平台对比）
- 用户需求画像
- 产品定位与定价策略
- PRD文档输出

### 2. Dev Agent (开发工程师)
- GPU租赁平台全栈开发
- 前端：用户界面、控制台、计费展示
- 后端：订单系统、资源调度、计费引擎
- 数据库：用户管理、GPU资源池、订单记录
- API集成：支付、监控、通知

### 3. UI Designer Agent (设计师)
- 平台整体视觉设计
- 用户控制台UI
- 移动端适配
- 组件库与设计系统
- 品牌VI设计

### 4. 运营Agent（数据分析师）
- 平台运营指标体系搭建
- 用户行为分析
- 转化率与留存率分析
- 收入与成本分析
- 数据看板与报表

### 5. 客服Agent
- 客户咨询自动回复
- 工单系统
- 常见问题库
- 投诉处理流程
- 满意度追踪

### 6. 安全Agent
- 平台安全架构设计
- 数据加密与隐私保护
- DDoS/CC防护
- 漏洞扫描与修复
- 合规审查（等保、数据出境等）

## 执行顺序建议
1. **第一阶段**: PM调研 → 输出PRD
2. **第二阶段**: UI设计 + Dev开发（并行）
3. **第三阶段**: 安全加固 + 客服体系搭建
4. **第四阶段**: 运营数据分析体系上线

## 技术栈参考
- 前端: React/Vue + TypeScript
- 后端: Node.js/Go/Python
- 数据库: PostgreSQL + Redis
- GPU调度: Kubernetes + NVIDIA GPU Operator
- 部署: Docker + CI/CD
