## 二、色彩体系

### 2.1 主色调

| 色彩 | 色值 | 用途 |
|------|------|------|
| **科技蓝** | `#3B82F6` | 主按钮、链接、高亮 |
| **深海蓝** | `#1E40AF` | 标题、强调文字 |
| **翡翠绿** | `#06D6A0` | 成功状态、CTA按钮、价格 |
| **暗色背景** | `#0A0E1A` | 页面背景、导航栏 |
| **卡片背景** | `#111827` | 卡片容器、模态框 |
| **边框色** | `#1E293B` | 分割线、边框 |

### 2.2 功能色

| 色彩 | 色值 | 用途 |
|------|------|------|
| 成功 | `#10B981` | 运行中、完成 |
| 警告 | `#F59E0B` | 待处理、注意 |
| 危险 | `#EF4444` | 错误、停止、删除 |
| 信息 | `#06B6D4` | 提示、帮助 |

### 2.3 渐变
- **主渐变**: `linear-gradient(135deg, #3B82F6, #06D6A0)`
- 用于：Hero标题、Logo背景、CTA按钮

## 四、图标规范

### 4.1 风格
- 线性图标 (2px stroke)
- 圆角端点
- 24x24 标准尺寸
- 单色填充（跟随文字颜色）

### 4.2 核心图标清单
| 图标 | 用途 |
|------|------|
| ⚡ 闪电 | 算力、快速 |
| 🖥️ 显示器 | 控制台 |
| 💰 钱包 | 计费/余额 |
| 📊 图表 | 监控/数据 |
| ⚙️ 齿轮 | 设置 |
| 🔑 钥匙 | SSH/认证 |
| 📱 手机 | 移动端 |
| 🔒 锁 | 安全 |
