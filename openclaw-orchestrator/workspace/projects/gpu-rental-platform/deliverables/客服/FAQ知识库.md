## 一、计费相关

### Q1: 如何充值？支持哪些支付方式？
**A**: 登录控制台 → 账单中心 → 点击"充值"按钮。支持以下支付方式：
- 支付宝（即时到账）
- 微信支付（即时到账）
- 银行转账（1-3工作日到账）
- 企业对公转账（1-3工作日到账）

最低充值金额：¥10

### Q2: 按小时计费如何计算？不足一小时怎么算？
**A**: 我们采用**按秒计费**，精确到秒。
- 公式：费用 = 运行秒数 × 每小时单价 / 3600
- 例如：A100 80G（¥6.00/小时）运行 30 分钟 = 6.00 × 1800 / 3600 = ¥3.00

### Q3: 余额不足时实例会怎样？
**A**: 
1. 余额低于 1 小时费用时，发送余额预警通知
2. 余额耗尽时，实例自动停止（不会立即释放）
3. 停止的实例保留 7 天，充值后可重新启动
4. 7 天后未充值，实例自动释放，数据不再保留

### Q4: 如何申请发票？
**A**: 账单中心 → 发票管理 → 申请发票
- 电子发票：24小时内开具，发送至注册邮箱
- 纸质发票：3-5工作日寄出，邮费自理
- 发票类型：增值税普通发票 / 增值税专用发票（需企业认证）

### Q5: 包月和按量付费哪个更划算？
**A**: 
- **短期使用**（<15天/月）：按量付费更灵活
- **长期使用**（>15天/月）：包月更划算，享约7折优惠
- **包年**：最大优惠约6折，适合稳定业务线

## 二、GPU实例使用

### Q6: 如何创建第一个GPU实例？
**A**: 
1. 登录控制台 → 点击"新建实例"
2. 选择GPU型号（如 A100 80G）
3. 选择预装镜像（如 Ubuntu 22.04 + PyTorch）
4. 选择SSH密钥（首次需先创建）
5. 确认订单 → 60秒内实例就绪

### Q7: 支持哪些操作系统和镜像？
**A**: 预装镜像包括：
- Ubuntu 22.04 + CUDA 12.2 + PyTorch 2.1
- Ubuntu 22.04 + CUDA 12.2 + TensorFlow 2.15
- Ubuntu 22.04 + 纯净系统（自行安装）
- 后续将支持自定义镜像上传

### Q8: 如何通过SSH连接到实例？
**A**: 
1. 实例列表 → 点击"SSH"按钮
2. 复制连接信息：`ssh root@<IP地址>`
3. 本地终端执行连接命令
4. 或使用控制台内置的Web Terminal

### Q9: 如何上传/下载数据？
**A**: 
- **SCP**: `scp file.txt root@<IP>:/root/`
- **SFTP**: 使用 FileZilla 等工具连接
- **rsync**: `rsync -avz ./data/ root@<IP>:/root/data/`
- 大文件建议使用数据盘挂载功能

### Q10: 实例可以挂载数据盘吗？
**A**: V2.0版本将支持数据盘挂载。当前版本可使用：
- 实例内置存储（释放后数据丢失）
- 外部对象存储（如阿里云OSS、腾讯云COS）

## 三、故障排查

### Q11: 实例启动失败怎么办？
**A**: 
1. 检查余额是否充足（至少需 2 小时费用）
2. 检查GPU型号是否有库存
3. 查看实例状态是否为"错误"
4. 如仍无法解决，提交工单并提供实例ID

### Q12: GPU识别不到怎么处理？
**A**: 
1. 执行 `nvidia-smi` 检查GPU状态
2. 确认驱动版本与CUDA版本匹配
3. 尝试重启实例
4. 如问题持续，提交工单

### Q13: 网络不通如何排查？
**A**: 
1. 检查实例状态是否为"运行中"
2. 确认IP地址正确
3. 本地执行 `ping <IP地址>` 测试连通性
4. 检查本地防火墙/安全组设置
5. 提交工单并提供traceroute结果

### Q14: 实例卡顿/响应慢怎么解决？
**A**: 
1. 执行 `nvidia-smi` 检查GPU利用率
2. 执行 `htop` 检查CPU/内存使用
3. 检查是否有其他进程占用资源
4. 必要时重启实例或升级GPU型号

## 四、账户与安全

### Q15: 如何修改密码？
**A**: 设置中心 → 个人信息 → 修改密码
- 需要输入当前密码
- 新密码至少8位，包含大小写字母和数字

### Q16: API Token如何管理？
**A**: 设置中心 → API Token管理
- 创建Token：填写名称、有效期
- Token仅在创建时显示一次，请妥善保存
- 可随时撤销Token
- 建议定期更换Token

### Q17: 如何设置子账号权限？
**A**: V2.0版本将支持子账号和RBAC权限管理。

### Q18: 注册送¥100体验金如何使用？
**A**: 
- 新用户注册自动到账
- 可用于任何GPU实例消费
- 有效期30天
- 不可提现、不可转赠
- 体验金用完后可充值继续使用

## 五、退款政策

### Q19: 可以退款吗？
**A**: 
- 体验金不可退款
- 充值余额可申请退款（扣除已使用部分）
- 退款申请需在充值后30天内提出
- 退款原路返回，3-5工作日到账
